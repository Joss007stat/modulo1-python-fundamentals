# Proyecto Módulo 1 – Fundamentos

## Descripción
Aplicación desarrollada en Streamlit que integra conceptos de programación como variables, condicionales, estructuras de datos, funciones, programación funcional y Programación Orientada a Objetos.

## Estructura
- aplicacion.py
- Funciones.py
- requirements.txt
- README.md

## Instrucciones de ejecución

1. Instalar dependencias:
   pip install -r requirements.txt

2. Ejecutar la aplicación:
   streamlit run aplicacion.py
